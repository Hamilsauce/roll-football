# Assets Directory

This directory contains game assets including images and sound effects.

## Image Assets

Place the following image files here (or generate them):
- `icon.png` - App icon (1024x1024 recommended)
- `splash.png` - Splash screen image (1242x2436 recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 recommended)
- `favicon.png` - Web favicon (48x48 recommended)

## Sound Assets

Place sound effect files in the `sounds/` directory:
- `touchdown.mp3` or `touchdown.wav` - Touchdown sound
- `turnover.mp3` or `turnover.wav` - Interception/fumble sound
- `gain.mp3` or `gain.wav` - Positive yardage sound
- `incomplete.mp3` or `incomplete.wav` - Incomplete/no gain sound

### Sound File Recommendations
- Format: MP3 or WAV
- Sample Rate: 44.1kHz
- Bit Rate: 128kbps for MP3
- Duration: 1-3 seconds
- Keep file sizes small for mobile performance

## Generating Placeholder Assets

You can use online tools or create simple colored squares as placeholders:
- Use any image editor to create simple icon images
- Use royalty-free sound effects from sites like freesound.org or zapsplat.com
